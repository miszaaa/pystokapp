# PyStokAPP
## Example app using Docker to run:
### Django
### Redis
### Celery
### Nginx
### Postgres
### Memcached
### Elastic


## Running project:
Before starting project change paths in docker-compose.yml from
```
/home/mwawrzyniak/.virtualenvs/pystok-docker/pystokapp/
```
to directory where project was cloned and run
```
docker-compose up -d
```
